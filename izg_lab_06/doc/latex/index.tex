Vašim úkolem je naprogramovovat vykreslování toho, co vám na cvičení zadá cvičí. Budete programovat pouze v jazyce G\+L\+SL. Všechna kreslená data budou procedurální -\/ vypočítaná za běhu. Cílem cvičení je seznámit se s jazykem G\+L\+SL v Open\+GL / Vulkan.\hypertarget{index_co}{}\doxysection{Které soubory editujete\+:}\label{index_co}
shaders/student.\+glsl\hypertarget{index_teorie}{}\doxysection{Teorie}\label{index_teorie}
G\+L\+SL je jazyk pro gramování shaderů. Shader a shader program je kód běžící na grafických kartách. Účelem shaderů je hýbat s objekty, obarvovat je, vytvářet efekty, počítat fyziku, ... Shaderů je několik druhů\+: 
\begin{DoxyItemize}
\item {\bfseries{Vertex Shader}} -\/ slouží pro práci s vrcholy (vertexy). Obvykle násobí vrcholy maticemi. 
\item {\bfseries{Fragment Shader}} -\/ též známý jako pixel shader. Slouží pro práci s fragmenty. Obvykle je obarvuje. 
\item Geometry Shader -\/ pracuje s primitivy. 
\item Teselační Shadery -\/ dva shadery sloužící pro teselování primitiv. 
\item Compute Shadery -\/ pro obecné výpočty, třeba fyziky. 
\item Další, nestandardní shadery, různá rozšíření (raytracing, primitive shadery ...) 
\end{DoxyItemize}

\hypertarget{index_jak}{}\doxysection{Jak na to\+:}\label{index_jak}
Archiv obsahuje zkopilované binárky pro Linux a pro Windows (izg\+\_\+lab06\+\_\+linux.\+bin, izg\+\_\+lab06\+\_\+windows.\+exe).~\newline
 Pokud vám fungují, není potřeba nic kompilovat!~\newline
 Pokud je nemůžete použít, je potřeba cvičení zkompilovat, viz \mbox{\hyperlink{index_comp}{Kompilace\+:}}.~\newline
 Aplikace vykresluje trojúhelníky (volá se příkaz gl\+Draw\+Arrays).~\newline
 A vyvolává se vertex a fragment shader v souboru shaders/student.\+glsl ~\newline
 Soubor student.\+glsl obsahuje pomocí preprocesoru vytyčené sekce vertex a fragment shaderu.~\newline
 Vaším úkolem je vykreslit zadání, které určí cvičící.~\newline
 Máte k dispozici pouze číslo vrcholu\+: \char`\"{}gl\+\_\+\+Vertex\+I\+D\char`\"{}.~\newline
 Na základě tohoto čísla vytvořte/vypočítejte vrcholy a barvy trojúhelníků tak, aby se vykreslil požadovaný objekt.~\newline
 Inspirujte se přiloženými shadery, které vykreslují různé příklady (složka shaders/).~\newline
 Informace k jazyce G\+L\+SL najdete ve specifikaci\+: \href{https://www.khronos.org/registry/OpenGL/specs/gl/GLSLangSpec.4.60.pdf}{\texttt{ G\+L\+SL}}~\newline
 Když uděláte chybu, vyskočí message box s chybou.~\newline
 Pamatujte, vertex shader se pouští pro každý vrchol trojúhelníků, jeho cílem je rozmístit vrcholy.~\newline
 Fragment shader se pouští nad každým fragmentem. Měl by počítat barvu.~\newline
 Aplikace se pokouší vykreslit velké množství trojúhelníků, vám stačí pouze pár, zbylé můžete \char`\"{}zdegenerovat\char`\"{} do nuly (gl\+\_\+\+Position=vec4(0)).~\newline
